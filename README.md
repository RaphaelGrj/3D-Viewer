# 3D-Viewer
🇫🇷 [FR] Studio 3D Figurine & Kitbash Tool
Transformez votre navigateur en un véritable studio de préparation. Cet outil "couteau suisse" est conçu spécifiquement pour les passionnés d'impression 3D et les peintres de figurines.

Télécharger sur GitHub :

https://github.com/RaphaelGrj/3D-Viewer.git

Fonctionnalités clés :

🛠 Kitbash & Assemblage : Importez plusieurs fichiers .STL, déplacez-les et ajustez-les avec précision (sliders ou valeurs numériques) pour créer vos propres conversions.

💡 Simulateur d'Éclairage (NMM/OSL) : Placez des lumières zénithales ou des sources colorées (OSL) pour visualiser les ombres et les reflets avant de peindre.

📏 Règle de Mesure : Mesurez n'importe quelle partie de vos modèles en mm d'un simple clic pour vérifier l'échelle.

⛰ Générateur de Socles : Créez des socles instantanément.

💾 Export STL : Téléchargez votre assemblage final prêt pour votre slicer.

Utilisation : Ouvrez simplement le fichier .html dans n'importe quel navigateur (Chrome, Firefox, etc.). Aucun abonnement, aucune installation.

🇬🇧 [EN] 3D Mini Studio & Kitbash Tool
Turn your browser into a professional prep studio. This "Swiss Army Knife" utility is designed specifically for 3D printing enthusiasts and miniature painters.

Download on GitHub :

https://github.com/RaphaelGrj/3D-Viewer.git

Key Features:

🛠 Easy Kitbashing: Import multiple .STL files, transform them with mm-precision (using sliders or direct input) to create your own custom conversions.

💡 Lighting Simulator (NMM/OSL): Place Zenithal lights or magical colored glows (OSL) to visualize highlights and shadows before grabbing your brushes.

📏 Measurement Ruler: Measure any part of your models in mm with a simple click-to-click tool to ensure perfect scaling.

⛰ Base Generator: Create bases in one click.

💾 STL Export: Download your finished assembly as a single file, ready for your slicer.

How to use: Simply open the .html file in any web browser. No subscription, no installation, 100% standalone.
